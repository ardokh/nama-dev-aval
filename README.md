# nama-dev-aval #
Teste de avaliação NAMA

This is a avaliation test, fetch a text file from html-form, parse it to
SQLite database and display all registers.

### Requirements ###
* Python3
* Flask
* SQLite

## Instructions ##
1. (Optional)
	1. `python3 -m venv nama-penv && source nama-penv/bin/activate`
2. `python3 -m pip install flask`
3. `git clone https://github.com/ardokh/nama-dev-aval`
4. `cd nama-dev-aval`
5. `python3 app.py`

6. Upload file on url `localhost:9000`
